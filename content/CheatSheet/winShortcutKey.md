# Windows 11 常用快捷键速查表 🪟

> 按功能分类，方便快速查找和使用

## 🔧 系统通用快捷键



| 快捷键                 | 功能说明                             |
| ---------------------- | ------------------------------------ |
| `Win`                  | 打开/关闭开始菜单                    |
| `Win + A`              | 打开快速设置（网络、音量、亮度等）   |
| `Win + B`              | 聚焦系统托盘区域                     |
| `Win + D`              | 显示/隐藏桌面（最小化/恢复所有窗口） |
| `Win + E`              | 打开文件资源管理器                   |
| `Win + I`              | 打开系统设置                         |
| `Win + L`              | 锁定电脑                             |
| `Win + M`              | 最小化所有窗口                       |
| `Win + Shift + M`      | 恢复最小化的窗口                     |
| `Win + R`              | 打开"运行"对话框                     |
| `Win + S` / `Win + Q`  | 打开搜索                             |
| `Win + T`              | 循环切换任务栏应用                   |
| `Win + X`              | 打开快捷菜单（类似右键开始按钮）     |
| `Win + Z`              | 打开贴靠布局（窗口分屏）             |
| `Win + .` 或 `Win + ;` | 打开表情符号面板                     |
| `Win + V`              | 打开剪贴板历史记录                   |
| `Win + P`              | 打开投影模式选择                     |
| `Win + Ctrl + D`       | 创建新虚拟桌面                       |
| `Win + Ctrl + F4`      | 关闭当前虚拟桌面                     |
| `Win + Ctrl + ←/→`     | 切换虚拟桌面                         |
| `Alt + F4`             | 关闭当前窗口/程序                    |
| `Alt + Tab`            | 在打开的应用间切换                   |
| `Alt + Esc`            | 按顺序循环切换窗口                   |
| `Ctrl + Shift + Esc`   | 直接打开任务管理器                   |
| `Win + Shift + S`      | 截图（区域/窗口/全屏）               |
| `Print Screen`         | 全屏截图（可设置直接保存到剪贴板）   |

## 🪟 窗口管理与贴靠布局



| 快捷键                  | 功能说明                               |
| ----------------------- | -------------------------------------- |
| `Win + ←` / `→`         | 将窗口贴靠到屏幕左/右半侧              |
| `Win + ↑` / `↓`         | 最大化 / 还原或最小化窗口              |
| `Win + Shift + ←` / `→` | 将窗口移动到另一个显示器               |
| `Win + Home`            | 最小化/恢复除当前窗口外的所有窗口      |
| `Win + Tab`             | 打开任务视图（查看所有窗口和虚拟桌面） |
| `F11`                   | 全屏/退出全屏（部分应用支持）          |

## 📁 文件资源管理器快捷键



| 快捷键             | 功能说明                                                     |
| ------------------ | ------------------------------------------------------------ |
| `Ctrl + T`         | 添加新标签(在同一个窗口打开主文件夹)                         |
| `Ctrl + N`         | 在新窗口中打开当前文件夹                                     |
| `Ctrl + W`         | 关闭当前资源管理器聚焦所在的标签页, 若只有一个标签页,则关闭窗口 |
| `Ctrl + Shift + N` | 新建文件夹                                                   |
| `Alt + ↑`          | 返回上一级文件夹                                             |
| `Alt + ←` / `→`    | 后退 / 前进导航历史                                          |
| `Alt + D`          | 聚焦地址栏(与腾讯电脑管家中的截图工具快捷键冲突)             |
| `Ctrl + F`         | 聚焦搜索框                                                   |
| `F2`               | 重命名选中文件/文件夹                                        |
| `F5`               | 刷新当前视图                                                 |
| `Ctrl + Shift + E` | 显示当前文件夹的上级目录树                                   |
| `Alt + Enter`      | 打开选中项目的属性                                           |
| `Ctrl + Tab`         | 切换到**下一个**标签页                 |
| `Ctrl + Shift + Tab` | 切换到**上一个**标签页                 |
| `Ctrl + 数字 (1-9)`  | 直接跳转到第 N 个标签页                |

## ⌨️ 文本编辑通用快捷键



| 快捷键                        | 功能说明                         |
| ----------------------------- | -------------------------------- |
| `Ctrl + C` / `V` / `X`        | 复制 / 粘贴 / 剪切               |
| `Ctrl + Z` / `Y`              | 撤销 / 重做                      |
| `Ctrl + A`                    | 全选                             |
| `Ctrl + F`                    | 查找                             |
| `Ctrl + H`                    | 替换                             |
| `Ctrl + S`                    | 保存                             |
| `Ctrl + P`                    | 打印                             |
| `Ctrl + B` / `I` / `U`        | 加粗 / 斜体 / 下划线（部分应用） |
| `Ctrl + ←` / `→`              | 按单词移动光标                   |
| `Ctrl + Backspace` / `Delete` | 删除前/后一个单词                |
| `Shift + 方向键`              | 选中文本                         |
| `Ctrl + Shift + 方向键`       | 按单词选中文本                   |

## 🌐 浏览器通用快捷键（Edge/Chrome等）



| 快捷键                               | 功能说明               |
| ------------------------------------ | ---------------------- |
| `Ctrl + T` / `W`                     | 新建 / 关闭标签页      |
| `Ctrl + Shift + T`                   | 恢复最近关闭的标签页   |
| `Ctrl + Tab` / `Ctrl + PgUp`         | 切换到下一个标签页     |
| `Ctrl + Shift + Tab` / `Ctrl + PgDn` | 切换到上一个标签页     |
| `Ctrl + 1~8`                         | 切换到指定位置的标签页 |
| `Ctrl + 9`                           | 切换到最后一个标签页   |
| `Ctrl + L` / `Alt + D`               | 聚焦地址栏             |
| `Ctrl + R` / `F5`                    | 刷新页面               |
| `Ctrl + Shift + R` / `Ctrl + F5`     | 强制刷新（忽略缓存）   |
| `Ctrl + D`                           | 添加当前页到收藏夹     |
| `Ctrl + H`                           | 打开历史记录           |
| `Ctrl + J`                           | 打开下载内容           |
| `F11`                                | 切换全屏模式           |

## 🎮 游戏与辅助功能



| 快捷键           | 功能说明                                   |
| ---------------- | ------------------------------------------ |
| `Win + G`        | 打开 Xbox Game Bar（游戏录制、性能监控等） |
| `Win + Alt + R`  | 开始/停止游戏录制（需Game Bar）            |
| `Win + Alt + G`  | 录制最近30秒游戏（需开启后台录制）         |
| `Win + +` / `-`  | 打开放大镜并放大/缩小                      |
| `Win + Esc`      | 退出放大镜                                 |
| `Win + Ctrl + C` | 打开/关闭颜色滤镜（需提前在设置中启用）    |
| `Win + Ctrl + N` | 打开讲述人设置                             |
| `Win + Ctrl + S` | 打开语音识别                               |
