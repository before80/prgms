з¬¬дёүеҚҒдәҢз« пјҡSSH иҝңзЁӢз®ЎзҗҶ

SSHпјҲSecure ShellпјүжҳҜLinuxз®ЎзҗҶе‘ҳзҡ„"з”ҹе‘Ҫзәҝ"вҖ”вҖ”жІЎжңүе®ғпјҢдҪ з®ЎдёҚдәҶжңҚеҠЎеҷЁгҖҒдј дёҚдәҶж–Үд»¶гҖҒзҷ»еҪ•дёҚдәҶиҝңзЁӢдё»жңәгҖӮ

SSHеҮәзҺ°д№ӢеүҚпјҢиҝңзЁӢз®ЎзҗҶз”Ёзҡ„жҳҜTelnetе’ҢrloginпјҢдҪҶиҝҷдәӣеҚҸи®®жңүдёҖдёӘиҮҙе‘Ҫй—®йўҳпјҡжҳҺж–Үдј иҫ“гҖӮдҪ иҫ“е…Ҙзҡ„еҜҶз ҒеңЁзҪ‘з»ңдёҠ"иЈёеҘ”"пјҢи°ҒжҲӘиҺ·и°ҒзңӢеҲ°гҖӮSSHз”ЁеҠ еҜҶжҠҖжңҜи§ЈеҶідәҶиҝҷдёӘй—®йўҳпјҢжүҖжңүж•°жҚ®йғҪз»ҸиҝҮеҠ еҜҶпјҢи®©й»‘е®ўж— д»ҺдёӢжүӢгҖӮ

жң¬з« й…ҚеҘ—и§Ҷйў‘пјҡSSHеӯҰдёҚдјҡпјҢжңҚеҠЎеҷЁз®ЎзҗҶе°ұжҳҜз©әи°ҲгҖӮжң¬з« и®©дҪ д»Һ"еҗ¬иҜҙиҝҮSSH"еҲ°"SSHиҖҒеҸёжңә"гҖӮ

32.1 SSH з®Җд»ӢпјҡSecure Shell е®үе…ЁиҝңзЁӢзҷ»еҪ•

SSHпјҲSecure ShellпјҢе®үе…ЁеӨ–еЈіеҚҸи®®пјүжҳҜдёҖз§ҚеҠ еҜҶзҡ„зҪ‘з»ңеҚҸи®®пјҢз”ЁдәҺеңЁдёҚе®үе…Ёзҡ„зҪ‘з»ңдёҠжҸҗдҫӣе®үе…Ёзҡ„иҝңзЁӢзҷ»еҪ•е’Ңе…¶д»–зҪ‘з»ңжңҚеҠЎгҖӮ

32.1.1 еҠ еҜҶйҖҡдҝЎ

SSHзҡ„ж ёеҝғжҳҜеҠ еҜҶгҖӮжүҖжңүдј иҫ“зҡ„ж•°жҚ®вҖ”вҖ”еҢ…жӢ¬дҪ иҫ“е…Ҙзҡ„еҜҶз ҒгҖҒдҪ жү§иЎҢзҡ„е‘Ҫд»ӨгҖҒдҪ дј иҫ“зҡ„ж–Үд»¶вҖ”вҖ”е…ЁйғЁиў«еҠ еҜҶгҖӮж”»еҮ»иҖ…еҚідҪҝеңЁзҪ‘з»ңдёҠжҠ“еҢ…пјҢд№ҹеҸӘиғҪзңӢеҲ°дёҖе Ҷд№ұз ҒгҖӮ

SSHж”ҜжҢҒеӨҡз§ҚеҠ еҜҶз®—жі•пјҡAESгҖҒ3DESгҖҒBlowfishгҖҒChaCha20зӯүгҖӮзҺ°д»ЈSSHй»ҳи®ӨдҪҝз”ЁAES-256-CTRжҲ–ChaCha20-Poly1305пјҢе®үе…ЁжҖ§жһҒй«ҳгҖӮ

32.1.2 з«ҜеҸЈпјҡ22

SSHй»ҳи®Өзӣ‘еҗ¬TCPз«ҜеҸЈ22гҖӮиҝҷдёӘз«ҜеҸЈжҳҜдә’иҒ”зҪ‘дёҠжңҖ"з№Ғеҝҷ"зҡ„з«ҜеҸЈд№ӢдёҖвҖ”вҖ”еӨ§еӨҡж•°жңҚеҠЎеҷЁз®ЎзҗҶе‘ҳйғҪз”Ёе®ғжқҘз®ЎзҗҶжңҚеҠЎеҷЁгҖӮ

е®үе…Ёе»әи®®пјҡжҠҠSSHз«ҜеҸЈж”№жҲҗйқһ22з«ҜеҸЈпјҢеҸҜд»ҘеҮҸе°‘еӨ§йҮҸзҡ„иҮӘеҠЁеҢ–жҡҙеҠӣз ҙи§Јж”»еҮ»гҖӮж”»еҮ»иҖ…йҖҡеёёеҸӘжү«жҸҸ22з«ҜеҸЈпјҢж”№з«ҜеҸЈдёҚдјҡи®©дҪ з»қеҜ№е®үе…ЁпјҲж”»еҮ»иҖ…еҸҜд»Ҙжү«жҸҸжүҖжңүз«ҜеҸЈпјүпјҢдҪҶдјҡжҳҫи‘—еҮҸе°‘еҫҲеӨҡ"еҷӘйҹі"гҖӮ

рҹ”’ жӣҙеҘҪзҡ„е®үе…ЁжҺӘж–Ҫпјҡ

  1. зҰҒз”ЁеҜҶз Ғзҷ»еҪ•пјҢеҸӘз”ЁеҜҶй’Ҙи®ӨиҜҒпјҲжңҖе®үе…Ёпјү
  2. дҪҝз”ЁFail2banиҮӘеҠЁе°ҒзҰҒжҡҙеҠӣз ҙи§ЈIP
  3. й…ҚзҪ®йҳІзҒ«еўҷеҸӘе…Ғи®ёзү№е®ҡIPи®ҝй—®SSH
  4. дҪҝз”Ёи·іжқҝжңәпјҲBastion HostпјүжҲ–VPN
  5. е®ҡжңҹжӣҙж–°SSHзүҲжң¬пјҢдҝ®еӨҚе®үе…ЁжјҸжҙһ

32.2 SSH е·ҘдҪңеҺҹзҗҶ

SSHзҡ„е®үе…ЁжҖ§жқҘиҮӘдәҺе®ғзҡ„еҠ еҜҶжңәеҲ¶е’ҢеҜҶй’ҘдәӨжҚўиҝҮзЁӢгҖӮж•ҙдёӘиҝһжҺҘе»әз«ӢиҝҮзЁӢеҲҶдёәеҮ дёӘйҳ¶ж®өгҖӮ

32.2.1 еҜҶй’ҘдәӨжҚў

SSHиҝһжҺҘе»әз«Ӣзҡ„з¬¬дёҖжӯҘжҳҜеҜҶй’ҘдәӨжҚўпјҲKey ExchangeпјүгҖӮе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁеҚҸе•ҶзЎ®е®ҡдҪҝз”Ёе“Әз§ҚеҠ еҜҶз®—жі•пјҢе№¶дәӨжҚўдёҙж—¶дјҡиҜқеҜҶй’ҘгҖӮ

еҜҶй’ҘдәӨжҚўз®—жі•пјҲKEXпјүпјҡECDHEпјҲжӨӯеңҶжӣІзәҝDiffie-HellmanпјүгҖҒDHпјҲDiffie-HellmanпјүзӯүгҖӮиҝҷдәӣз®—жі•и®©еҸҢж–№еңЁдёҖжқЎдёҚе®үе…Ёзҡ„йҖҡйҒ“дёҠпјҢ“еҚҸе•Ҷ"еҮәдёҖдёӘеҸӘжңүеҸҢж–№зҹҘйҒ“зҡ„е…ұдә«еҜҶй’ҘпјҢиҖҢж”»еҮ»иҖ…ж— жі•жҺЁз®—еҮәжқҘгҖӮ

32.2.2 еҜ№з§°еҠ еҜҶ

еҜҶй’ҘдәӨжҚўе®ҢжҲҗеҗҺпјҢе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁдјҡз”ҹжҲҗдёҖдёӘдјҡиҜқеҜҶй’ҘпјҲSession KeyпјүпјҢз”ЁдәҺеҜ№з§°еҠ еҜҶеҗҺз»ӯзҡ„жүҖжңүйҖҡдҝЎж•°жҚ®гҖӮ

еҜ№з§°еҠ еҜҶзҡ„зү№зӮ№пјҡеҠ еҜҶе’Ңи§ЈеҜҶз”ЁеҗҢдёҖжҠҠй’ҘеҢҷпјҢйҖҹеәҰеҝ«пјҢйҖӮеҗҲеҠ еҜҶеӨ§йҮҸж•°жҚ®гҖӮ

32.2.3 йқһеҜ№з§°еҠ еҜҶ

еңЁеҜҶй’ҘдәӨжҚўиҝҮзЁӢдёӯпјҢSSHдҪҝз”ЁйқһеҜ№з§°еҠ еҜҶпјҲе…¬й’ҘеҠ еҜҶгҖҒз§Ғй’Ҙи§ЈеҜҶпјүжқҘе®үе…Ёең°дј иҫ“дјҡиҜқеҜҶй’ҘгҖӮ

жңҚеҠЎеҷЁе’Ңе®ўжҲ·з«Ҝеҗ„иҮӘжңүдёҖеҜ№еҜҶй’Ҙпјҡ

  • жңҚеҠЎеҷЁеҜҶй’ҘеҜ№пјҡжңҚеҠЎеҷЁдёҠзҡ„дё»жңәе…¬й’Ҙ/з§Ғй’ҘпјҢз”ЁдәҺйӘҢиҜҒжңҚеҠЎеҷЁиә«д»Ҫ
  • е®ўжҲ·з«ҜеҜҶй’ҘеҜ№пјҲеҸҜйҖүпјүпјҡз”ЁжҲ·зҡ„е…¬й’Ҙ/з§Ғй’ҘпјҢз”ЁдәҺе…ҚеҜҶз Ғзҷ»еҪ•
